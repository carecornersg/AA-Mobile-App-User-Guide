# AA Mobile App User Guide

Welcome to the **Appropriate Adult (AA) Mobile App** User Guide. This documentation will help you understand and use the CAS mobile application.

---

## What is the AA Mobile App?

The AA Mobile App allows Appropriate Adults to:

- **Receive Case Broadcasts**: Get notified of new case assignments
- **Accept/Acknowledge Cases**: Respond to case broadcasts in real-time
- **Manage Profile**: Update availability, preferences, and contact information
- **View History**: See past assignments and case details
- **Earn Badges**: Track achievements and milestones

## Getting Started

1. **Download the App** from App Store (iOS) or Play Store (Android)
2. **Login**
3. **Verify** with Email OTP (6 digits)
4. **Enable Biometric Login** (Recommended)
5. **Start Receiving Cases!**
